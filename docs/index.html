易录屏

一个使用 Electron 开发的简洁高效的屏幕录制工具

快速录制屏幕活动并保存为视频文件，轻量级，功能精简，启动快速，占用资源少

立即下载 查看源码

功能特点

一键录制

一键开始录制整个屏幕，多屏幕状态下可自由选择要录制屏幕

实时计时

实时显示录制时长，掌握录制进度

暂停/恢复

支持暂停/恢复录制功能，灵活控制录制过程

时长限制

支持设置最大录制时长，到达时间后自动停止录制

视频预览

录制完成后提供视频预览功能，即时查看录制效果

自动最小化

录制开始后自动最小化窗口，不影响录制内容

多格式保存

支持将录制内容保存为 WebM/MP4 格式

跨平台支持

支持 Windows/macOS/Linux 多平台使用

从源码构建

git clone https://github.com/SignitDoc/yi-recorder.git
cd yi-recorder
npm install
npm start

构建应用（生成安装程序和便携版）：

npm run build-win

构建后的文件会生成在 dist 目录下

使用说明

1

启动应用程序

双击安装后的应用图标启动易录屏

2

设置录制选项

可选择是否录制系统声音，并设置最大录制时长（0 表示无限制，默认为 60 分钟）

3

开始录制

点击"开始录制"按钮开始录制屏幕，多屏幕状态下，选择要录制的屏幕

4

录制过程

应用将在倒计时 3 秒后自动最小化，录制继续在后台进行

5

暂停/继续

录制过程中，可以点击任务栏中的应用图标，使用"暂停录制"按钮暂停录制，再次点击继续录制

6

停止录制

需要停止录制时，点击任务栏中的应用图标，然后点击"停止录制"

7

预览和保存

录制结束后会显示视频预览，可以查看录制内容，点击"保存录制"，选择保存位置即可生成视频文件
